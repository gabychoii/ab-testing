02 - Hypotheses
GOAL: Create and justify null and alternative hypotheses for the following three categories we are testing: Schedule an appointment with Adam Ng, MD at Morristown Medical Center on April 23, 2024
Misclick Rate
DEFINITION
The frequency with which users click something else on the page before finding the correct button for the task
NULL HYPOTHESIS
There will be no difference between the misclick rate in Version A and the misclick rate in Version B
ALTERNATIVE HYPOTHESIS
There will be a difference in the misclick rate in Version A versus Version B (Version A will have a higher misclick rate than Version B)
PREDICTION + ALT HYPOTHESIS REASONING
I hypothesize that the null hypothesis will be rejected. Given the definition of misclick rate, we hypothesize that the design change made in Version B (increased contrast in color and change in flow of information) will make the task of booking an appointment easier for users, it logically follows that the misclick rate will likely be lower, and therefore not the same (rejecting the null hypothesis)
Time on Page
DEFINITION
The frequency with which users click something else on the page before finding the correct button for the task
NULL HYPOTHESIS
There will be no differnece in the amount of time users spend on a page (completing a task) using Version A versus Version B
ALTERNATIVE HYPOTHESIS
There will be a difference in the time users spent on page for Version A versus Version B (Users will spend more time on a page completing a task using Version A than using Version B)
PREDICTION + ALT HYPOTHESIS REASONING
I hypothesize that the null hypothesis will be rejected. Given the definition of time on page, if we hypothesize that the design change made in Version B (increased contrast in color and change in flow of information) will make the task of booking an appointment easier for users, it logically follows that users will complete the task more quickly, resulting in a reduced amount of time spent on the page.

Summary Statistics
The outputs of the data analysis performed on these datasets (pictured above) represnets the shape and patterm of the data. We can visualize the direction and magnitude of change graphically. In this case, the mean is the most important indicator of change between Version A and B, as it is an indicative measure of average performance when completing the task at hand - given the context of our measurements, our goal is to make a design change(s) that will reduce the difficulty/time of this task. Therefore, mean is a crucial measurement.
Count, which represents the size of the data set, is also important, as the larger the count, the more reliable the data becomes, as it represents a larger group of people. In this case, our count is 39, which is a relatively small data set - this tells us our data could be slightly biased.
Lastly, median is the data point that lies in the middle of the sorted set. This value is less important than mean because it can be influenced by outliers, but is generally representative of the average.
An example of analysis invoving this numbers can be seen with the misclick rate output and graph - we see that the misclick rate was the factor most significantly affected by the change in design from Version A to Version B. Thus, this tells us that our design changes may have assisted with button/information clarity.
04 - Conclusion
Conclusion
Misclick Rate
The difference between the misclick rate in Version A and Version B is statistically significant, and therefore we reject the null hypothesis. The misclick rate in Version A is higher than the mislick rate in version B. We can assume that the changes made to Version B (increased contrast and logic re-ordering) effectively made the task easier to complete.
Time on Page
The difference between the time users spent on page using Version A versus Version B is not statistically significant, and therefore we accept the null hypothesis. We can assume that the changes made to Version B (increased contrast and logic re-ordering) were not effective enough to reduce the time spent on the task.
Mouse Movement
The difference between the total distance in mouse movement on Version A and B is statistically significant, and therefore we reject the null hypothesis. The mouse movement total in Version A is higher than the mouse movement total in Version B. We can logically assume that the increased contrast and logical-reordering of Version B made the site functions clearer and the overall task more straightforward (resulting in reduced scrolling and mouse movement)
Overall, 2/3 metrics indicated statistically significant improvememnts in task efficiency after changes were made from Version A to Version B. Thus, we can conclude that the design choices were effective, but there is still room for improvement in design and further efficiency.
